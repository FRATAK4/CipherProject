## General info
<details>
Cipher is a simple command-line program written in Python that allows text encryption using shift values of 13 and 47, as well as saving and reading it from a file.
The user can choose one of five available options.
</details>

## Setup
1. Download or clone the repository.
2. Ensure that Python is installed.
3. Run the program:
   ```bash
    python src/main.py
   ```
3. Run the test:
   ```bash
   cd src
   python -m pytest
   ```
